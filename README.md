NoFap Takip, bağımlılıkla mücadele eden kullanıcılara yardımcı olmak için tasarlanmış bir Android uygulamasıdır.


## Özellikler



- 📅 Streak takibi
- 🏆 Başarı rozetleri
- 📝 Relapse geçmişi
- 💪 Motivasyon mesajları
- 🌙 Karanlık/Aydınlık tema desteği
